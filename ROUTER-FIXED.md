## 🐛 问题原因

**错误的代码**：
```javascript
model: selectedModel.replace('-', '.')  // 错误！
```

这把 `glm-4-flash` 转换成了 `glm-4.flash`（不存在的模型）

**正确的做法**：
```javascript
model: selectedModel  // 直接使用原始名称
```
