# 🔄 清除缓存并重新测试

## 方法1：强制刷新（推荐）

### 在手机浏览器：
1. 打开页面
2. 长按刷新按钮
3. 选择"清除缓存并硬性重新加载"

或：
1. 在浏览器设置中找到"清除缓存"
2. 清除缓存
3. 重新打开页面

## 验证是否是新版本

打开浏览器控制台（F12），应该看到：

**新版本日志**：
```
🔍 请求类型: image_gen
🎨 调用图像生成API...
📸 图片URL: https://...
```

**旧版本日志**：
```
🔀 智能路由: text → glm-4-flash
```

---

## 测试步骤

1. **清除缓存**（上述方法任选其一）

2. **访问页面**

3. **输入**：`画一只猫`

4. **查看控制台**：
   - 如果看到 `🎨 调用图像生成API` → 新版本 ✅
   - 如果看到 `🔀 智能路由: text` → 旧版本 ❌

5. **查看结果**：
   - 如果看到图片 → 成功 ✅
   - 如果看到文字 → 还在用旧版本 ❌
