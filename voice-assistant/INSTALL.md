# TOKI APP 安装指南

## 📱 安装方式

### 方式1：PWA安装（推荐，立即可用）

#### iPhone（iOS）
1. 用Safari打开：
   ```
   https://chaseqq123.github.io/toki-app/voice-assistant/
   ```

2. 点击底部"分享"按钮（方框向上箭头）

3. 向下滑动，点击"添加到主屏幕"

4. 点击右上角"添加"

5. ✅ 完成！主屏幕上会出现TOKI图标

#### Android手机
1. 用Chrome打开：
   ```
   https://chaseqq123.github.io/toki-app/voice-assistant/
   ```

2. 等待几秒，会弹出"安装TOKI"提示

3. 点击"安装"

4. ✅ 完成！APP已安装到手机

**如果没有弹出提示**：
1. 点击Chrome右上角三个点
2. 点击"添加到主屏幕"或"安装应用"

## 🔧 常见问题

### Q: iPhone上没有"添加到主屏幕"选项？
**A**: 
- 必须使用Safari浏览器
- 确保已经打开网页
- 网页需要完全加载完成

### Q: Android上没有安装提示？
**A**:
- 使用Chrome浏览器
- 等待几秒钟
- 或手动点击菜单 → "添加到主屏幕"

### Q: 安装后打开是白屏？
**A**:
- 检查网络连接
- 清除浏览器缓存重试
- 删除图标重新安装

### Q: 麦克风无法使用？
**A**:
- 检查浏览器权限设置
- iPhone: 设置 → Safari → 麦克风 → 允许
- Android: Chrome → 设置 → 网站设置 → 麦克风
