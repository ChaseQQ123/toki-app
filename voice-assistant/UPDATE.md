## 🔧 功能详解

### 语音合成原理

```javascript
// 使用Web Speech API
const utterance = new SpeechSynthesisUtterance(text);
utterance.lang = 'zh-CN';
utterance.rate = 1.0;  // 语速
utterance.pitch = 1.0;  // 音调
utterance.volume = 1.0; // 音量

window.speechSynthesis.speak(utterance);
```

**特点**：
- 完全免费
- 无需API
- 支持中文
- 可调节语速

---

### 视频通话原理

```javascript
// 使用WebRTC
navigator.mediaDevices.getUserMedia({
    video: true,
    audio: true
}).then(stream => {
    videoElement.srcObject = stream;
});
```

**特点**：
- 实时视频
- 完全免费
- 无需服务器（P2P）
- 支持摄像头/麦克风切换

## ⚠️ 常见问题

### 问题1：没有语音回答

**可能原因**：
- 浏览器不支持
- 语音列表未加载

**解决方法**：
1. 使用Chrome浏览器
2. 刷新页面重试
3. 检查浏览器设置中的语音权限

### 问题2：视频通话无法启动

**可能原因**：
- 未允许摄像头权限
- 设备无摄像头

**解决方法**：
1. 点击浏览器地址栏的锁图标
2. 找到"摄像头"和"麦克风"
3. 选择"允许"
4. 刷新页面
