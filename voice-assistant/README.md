## 🎯 使用方法

### 基本对话

1. 点击麦克风按钮 🎤
2. 开始说话（例如："你好"）
3. 等待AI回复
4. 自动语音播放

### 快捷操作

- 说"你好" → 打招呼
- 说"现在几点" → 查看时间
- 说"介绍一下自己" → 了解TOKI
- 说"今天天气" → 查看天气

## 🐛 常见问题

### 问题1：麦克风权限被拒绝

**解决方案**：
1. 点击浏览器地址栏左侧的锁图标
2. 找到"麦克风"选项
3. 选择"允许"
4. 刷新页面

### 问题2：语音识别无法启动

**解决方案**：
- 确保使用Chrome浏览器
- 检查网络连接（Web Speech API需要网络）
- 检查麦克风是否正常工作

### 问题3：没有声音

**解决方案**：
- 检查系统音量
- 检查浏览器是否静音
- 尝试刷新页面
