# TOKI 项目计划书

> One Token, All AI Models

---

## 📋 项目概述

### 项目名称
**TOKI - AI助手手机应用**

### 项目定位
面向普通用户的智能生活助手，通过Token套餐模式提供AI服务

### 核心理念
**不是AI技术展示，而是解决实际问题**

### 目标用户
- 年龄：18-45岁
- 特点：时间碎片化、价格敏感、追求效率
- 痛点：不会写文案、语言障碍、容易忘事、选择困难、知识焦虑

## 🎯 核心功能（MVP版本）

### 功能1：智能写作助手 ⭐⭐⭐⭐⭐

#### 功能描述
一键生成社交媒体文案、工作文档、邮件回复

#### 核心功能
- **社交媒体文案**
  - 朋友圈文案（配表情+标签）
  - 小红书笔记（带话题标签）
  - 抖音/快手文案

- **工作文档**
  - 工作邮件（商务/日常）
  - 周报/月报/年度总结
  - 会议纪要
  - 工作计划

- **日常写作**
  - 回复建议（领导/客户/朋友）
  - 投诉信/感谢信
  - 邀请函/通知

#### 使用场景示例

**场景1：朋友圈文案**
```
用户输入："今天吃了火锅"
TOKI输出：
"今天终于打卡了心心念念的火锅！🔥
热辣鲜香的汤底，配上新鲜食材，简直是冬日快乐源泉～
📍 地址：XXX火锅，强烈推荐！⭐️⭐️⭐️⭐️⭐️
#火锅打卡 #美食日记"
```

**场景2：工作邮件**
```
用户输入："回复客户说价格可以商量"
TOKI输出：
"尊敬的王总：

感谢您的来信！

关于您提到的价格问题，我们可以进一步沟通协商。
我们的产品经过严格质量把控，相信能为您带来超值体验。

方便的话，我们可以约个时间详细聊聊？

此致
敬礼

[你的名字]
[日期]"
```

#### 技术实现
- 接入TOKNM API（DeepSeek/Qwen模型）
- 针对中文社交平台优化prompt
- 自动添加表情符号和标签
- 提供多种风格选择（正式/轻松/幽默）

#### 差异化优势
- 针对中国社交平台优化（vs ChatGPT）
- 自动格式化和表情添加（vs 通用AI）
- 一键复制到剪贴板
- 本地保存历史记录

#### 开发优先级
**P0 - 最高优先级**

### 前端架构

#### 技术栈选择
```
方案A：React Native
- 优势：跨平台，一套代码iOS+Android
- 开发速度：快
- 性能：中等
- 推荐指数：⭐⭐⭐⭐⭐

- 优势：性能好，UI美观
- 学习曲线：陡峭
- 推荐指数：⭐⭐⭐⭐

方案C：原生开发
- 优势：性能最佳
- 开发成本：高
- 推荐指数：⭐⭐⭐
```

- 开发效率高
- 社区成熟
- 易于维护
