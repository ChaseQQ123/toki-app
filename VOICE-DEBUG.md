## 🐛 如果还是没声音

### 方法1：检查浏览器控制台

**电脑Chrome**：
1. 按 `F12` 打开开发者工具
2. 点击 `Console` 标签
3. 刷新页面
4. 查看日志

**正常日志**：
```
✅ TOKI语音助手已启动
🔊 语音合成状态: 可用
🗣️ 可用语音数量: 15
✅ 已选择中文语音: Google 普通话
🔊 播放测试语音: 语音功能已就绪
✅ 测试语音开始
✅ 测试语音结束
```

**异常日志**：
```
❌ 语音合成不可用
⚠️ 没有可用语音
❌ 测试语音失败: not-allowed
```

---

### 方法2：手动测试语音合成

**在浏览器控制台输入**：
```javascript
// 创建语音
const utterance = new SpeechSynthesisUtterance('你好，我是TOKI');
utterance.lang = 'zh-CN';

// 播放
speechSynthesis.speak(utterance);
```

**如果能听到声音** → 语音合成正常
**如果没声音** → 浏览器或系统设置问题

---

### 方法3：检查系统设置

**手机**：
1. 检查音量是否打开
2. 检查是否静音模式
3. 检查是否允许浏览器播放音频

**电脑**：
1. 检查音量混合器
2. 检查浏览器是否被静音
3. 检查系统音频输出

---

### 方法4：更换浏览器

**推荐浏览器**：
- ✅ **Chrome**（最佳支持）
- ✅ **Edge**（良好支持）
- ✅ **Safari**（iPhone推荐）

- 需要允许音频权限
- 某些手机需要点击页面才能播放
- 推荐使用Chrome浏览器

### iPhone手机
- 推荐使用Safari浏览器
- 需要允许音频权限
- 某些iOS版本有限制
